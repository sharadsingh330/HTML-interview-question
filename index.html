36.How many type of heading does html contain

HTML contains six types of headings, ranging from < h1> to < h6>. < h1> → Largest and most important heading (Main title).
< h2> → Second level heading (Subheading).
< h3> → Third level heading.
< h4> → Fourth level heading.
< h5> → Fifth level heading.
< h6> → Smallest heading, least important.
Example:
< h1>Main Title
< h2>Subheading
< h3>Section Title
< h4>Subsection
< h5>Smaller Heading
< h6>Smallest Heading
Each heading is displayed in decreasing size, with < h1> being the largest and < h6> the smallest.

85. What do you mean by MARKUP language and hypertext in HTML?

In HTML, "markup language" refers to a system of codes (called tags) used to define the structure and formatting of a web page, indicating how text, images, and other elements should be displayed, while "hypertext" refers to the ability to link from one part of a document to another, essentially creating clickable connections between different web pages within a website or across the internet; essentially, HTML is a markup language that enables hypertext functionality.

86. How are active links different from normal links?

An "active link" refers to a hyperlink that is currently being clicked on, meaning it's in the process of being activated, while a "normal link" is a standard hyperlink that appears on a webpage, but is not currently being clicked on; essentially, an active link is the momentary state of a link when you press the mouse button to click it, visually indicated by a color change or other visual cue depending on the website design.
